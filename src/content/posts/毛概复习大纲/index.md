# 毛概期末考试复习大纲

---

## 一、考试基本信息

### 1.1 题型与分值

| 题型           | 题量         | 每题分值 | 总分            |
| -------------- | ------------ | -------- | --------------- |
| 单选题         | 20题         | 1分      | 20分            |
| 多选题         | 10题         | 2分      | 20分            |
| 判断题         | 10题         | 1分      | 10分            |
| 材料题         | 2题（各2问） | 10分/问  | 40分            |
| 论述题         | 1题          | 10分     | 10分            |
| **合计** |              |          | **100分** |

### 1.2 出题范围

- **客观题（前50分）**：每章每节都有，全面覆盖
- **主观题（后50分）**：**仅出在导论、第一章、第二章、第四章、第六章**

### 1.3 答题技巧（老师强调）

- 多选题：错选、漏选均不得分，必须非常准确
- 材料题：每题有两问，每问10分。**必须写满80%以上的答题空间**，不能只写一两行。实在没话写就抄材料中的相关内容
- 论述题：10分，同样要展开写

### 【导论】马克思主义中国化时代化的历史进程与理论成果

#### 核心知识点

1. **马克思主义中国化时代化的提出**
   - 1938年10月，毛泽东在**党的六届六中全会**上作《论新阶段》报告，正式提出"马克思主义中国化"命题
2. **马克思主义中国化时代化的内涵**
   - 立足中国国情和时代特点，坚持把马克思主义基本原理**同中国具体实际相结合、同中华优秀传统文化相结合**（"两个结合"）
   - 深入研究和解决中国革命、建设、改革不同历史时期的实际问题
3. **马克思主义中国化时代化的历史进程（三次飞跃）**
   - **第一次飞跃**：毛泽东思想
   - **新的飞跃**：中国特色社会主义理论体系（邓小平理论、"三个代表"重要思想、科学发展观）
   - **新的飞跃**：习近平新时代中国特色社会主义思想
4. **理论成果及其关系**
   - 重大理论成果：毛泽东思想；邓小平理论；"三个代表"重要思想；科学发展观；习近平新时代中国特色社会主义思想
   - 教材主线：**马克思主义中国化时代化**
   - 各理论成果之间是**一脉相承又与时俱进**的关系
5. **中国特色社会主义理论体系**
   - 包括：邓小平理论、"三个代表"重要思想、科学发展观、习近平新时代中国特色社会主义思想
   - **不包括**毛泽东思想
   - 实现了马克思主义中国化时代化**新的飞跃**（2023版教材不再用"第二次历史性飞跃"的表述）

#### 必背简答题

- **马克思主义中国化时代化的内涵**
- **马克思主义中国化时代化的重大理论成果**

### 【第一章】毛泽东思想及其历史地位

#### 核心知识点

1. **毛泽东思想形成发展的历史条件**
   - 时代背景：战争与革命成为时代主题；1917年俄国十月革命胜利
   - 实践基础：中国共产党领导人民进行革命和建设的成功实践
2. **毛泽东思想形成与发展的阶段**

| 阶段     | 时期                             | 标志                                 | 代表著作                                                                                         |
| -------- | -------------------------------- | ------------------------------------ | ------------------------------------------------------------------------------------------------ |
| 萌芽     | 大革命时期（第一次国内革命战争） | 新民主主义革命基本思想的提出         | 《中国社会各阶级的分析》《湖南农民运动考察报告》                                                 |
| 初步形成 | 土地革命战争时期                 | 农村包围城市、武装夺取政权思想的提出 | 《中国的红色政权为什么能够存在？》《井冈山的斗争》《星星之火，可以燎原》《反对本本主义》         |
| 成熟     | 遵义会议后到抗日战争时期         | 系统阐述新民主主义革命理论           | 《矛盾论》《实践论》《〈共产党人〉发刊词》《中国革命和中国共产党》《新民主主义论》《论联合政府》 |
| 继续发展 | 解放战争时期和新中国成立后       | 社会主义革命和建设重要思想           | 人民民主专政理论、社会主义改造理论、正确处理人民内部矛盾理论                                     |

1. **毛泽东思想确立为指导思想**
   - **党的七大**（1945年）正式把毛泽东思想写入党章
2. **毛泽东思想的主要内容**
   - 新民主主义革命理论
   - 社会主义革命和社会主义建设理论
   - 革命军队建设和军事战略的理论
   - 政策和策略的理论
   - 思想政治工作和文化工作的理论
   - 党的建设理论
3. **毛泽东思想活的灵魂**（三个基本方面）
   - **实事求是**：一切从实际出发，理论联系实际，坚持在实践中检验真理和发展真理（是毛泽东思想的**精髓**）
   - **群众路线**：一切为了群众，一切依靠群众，从群众中来，到群众中去（党的根本工作路线）
   - **独立自主**：坚持独立思考，走自己的路，维护民族独立、捍卫国家主权
4. **毛泽东思想的历史地位**
   - 马克思主义中国化时代化的第一个重大理论成果
   - 中国革命和建设的科学指南
   - 中国共产党和中国人民宝贵的精神财富

#### 必背简答题

- **毛泽东思想的主要内容**
- **毛泽东思想的活的灵魂**
- **如何科学认识毛泽东思想的历史地位**（教材37页思考题第三题）

### 【第二章】新民主主义革命理论

#### 核心知识点

1. **近代中国国情**
   - 最基本的国情：**半殖民地半封建社会**
   - 主要矛盾：帝国主义和中华民族的矛盾、封建主义和人民大众的矛盾（其中帝国主义和中华民族的矛盾是**最主要的矛盾**）
2. **新民主主义革命总路线**
   - 1948年毛泽东在《在晋绥干部会议上的讲话》中完整表述：
   - **无产阶级领导的，人民大众的，反对帝国主义、封建主义和官僚资本主义的革命**
3. **革命对象（三座大山）**
   - 帝国主义（**首要对象**，是中国贫困落后和一切灾难祸害的**总根源**）
   - 封建主义
   - 官僚资本主义
4. **革命动力**
   - **无产阶级**：中国革命最基本的动力，领导力量
   - **农民阶级**：中国革命的主力军，贫农是最可靠的同盟军
   - **城市小资产阶级**：包括知识分子、小商人、手工业者、自由职业者，是无产阶级的可靠同盟者
   - **民族资产阶级**：也是中国革命的动力之一
5. **革命领导力量**
   - 无产阶级的领导权是中国革命的**中心问题**，也是新民主主义革命理论的**核心问题**
   - 区别新旧民主主义革命的**根本标志**：革命的领导权是掌握在无产阶级手中还是资产阶级手中
6. **革命性质与前途**
   - 性质：**新式的特殊的资产阶级民主主义革命**（不是无产阶级社会主义革命）
   - 新旧民主革命的共同点：都是**资产阶级民主主义革命**
   - 前途：**社会主义**（不是资本主义）
7. **"新"在哪里**
   - 领导阶级不同（无产阶级 vs 资产阶级）
   - 指导思想不同（马克思主义）
   - 属于世界无产阶级革命的一部分
   - 革命前途不同（社会主义）
8. **新民主主义基本纲领**
   - **政治纲领**：建立无产阶级领导的、以工农联盟为基础的、各革命阶级联合专政的新民主主义共和国
     - 国体：各革命阶级联合专政
     - 政体：民主集中制的人民代表大会制度
   - **经济纲领**：没收封建地主阶级的土地归农民所有；没收官僚资本归新民主主义国家所有（**具有双重性质**）；保护民族工商业（**极具特色的一项**）
   - **文化纲领**：无产阶级领导的人民大众的反帝反封建的文化，即**民族的、科学的、大众的**文化
9. **新民主主义革命道路**
   - **农村包围城市、武装夺取政权**
   - 根本在于处理好**土地革命、武装斗争、农村革命根据地建设**三者之间的关系
     - 土地革命：民主革命的基本内容
     - 武装斗争：中国革命的主要形式
     - 农村革命根据地：中国革命的战略阵地
10. **三大法宝**
    - **统一战线、武装斗争、党的建设**
    - 毛泽东在《〈共产党人〉发刊词》中提出
    - 关系：统一战线和武装斗争是战胜敌人的两个基本武器，统一战线是实行武装斗争的统一战线，武装斗争是统一战线的中心支柱，党的组织则是掌握这两个武器的英勇战士
11. **民族资产阶级的两面性**
    - **革命性**：遭受帝国主义、封建主义的压迫
    - **妥协性**：与帝国主义和封建主义有千丝万缕的联系
    - 对其政策：**既斗争又联合**，在斗争中求团结
12. **统一战线**
    - 两个联盟：工人阶级同农民、小资产阶级等其他劳动者的联盟（基本的、主要的）；工人阶级同可以合作的非劳动者的联盟
    - 必须坚持**独立自主**原则
    - 对资产阶级实行**又联合又斗争**的方针

#### 必背简答题

- **新民主主义革命的总路线**
- **新民主主义基本纲领的主要内容**
- **中国革命走农村包围城市、武装夺取政权道路的必要性及重要意义**
- **新民主主义革命的三大法宝及其相互关系**
- **土地革命、武装斗争、农村革命根据地建设三者之间的关系**

### 【第四章】社会主义建设道路初步探索的理论成果

#### 核心知识点

1. **《论十大关系》**
   - 1956年，毛泽东作《论十大关系》报告
   - 标志着党探索中国社会主义建设道路的**良好开端**
   - 明确提出**以苏为鉴**，独立自主地探索适合中国情况的社会主义建设道路
   - 基本方针：**调动一切积极因素，为社会主义事业服务**
   - 第一条讨论：重工业和轻工业、农业的关系
   - 前三条实际上明确提出了**中国的工业化道路问题**
2. **《关于正确处理人民内部矛盾的问题》**
   - 1957年2月，毛泽东作此报告，系统论述了社会主义社会矛盾的理论
3. **社会主义社会的基本矛盾**
   - 仍然是**生产关系和生产力之间、上层建筑和经济基础之间的矛盾**
   - 但与旧社会的基本矛盾具有**根本不同的性质和情况**（非对抗性的）
   - 可以通过社会主义制度本身的自我调整和自我完善得到解决
4. **社会主义社会两类不同性质的矛盾**
   - **敌我矛盾**：对抗性的，用**专政**的方法解决
   - **人民内部矛盾**：非对抗性的，用**民主**的方法解决（总方针）
   - **正确处理人民内部矛盾是社会主义国家政治生活的主题**
5. **处理人民内部矛盾的具体方针**
   - 政治思想领域：**"团结—批评—团结"**，坚持说服教育、讨论的方法
   - 物质利益/分配方面：统筹兼顾、适当安排
   - 人民群众和政府机关的矛盾：坚持民主集中制
   - 科学文化领域：**"百花齐放、百家争鸣"**
6. **党的八大（1956年）关于主要矛盾的论述**
   - 国内主要矛盾：人民对于建立先进的工业国的要求同落后的农业国的现实之间的矛盾，人民对于经济文化迅速发展的需要同当前经济文化不能满足人民需要的状况之间的矛盾
   - 党和国家的工作重点：集中力量发展生产力，**技术革命和社会主义建设**
7. **走中国工业化道路**
   - 主要是指**重工业和轻工业、农业的发展关系问题**
   - 强调走一条**有别于苏联**的中国工业化道路
8. **关于社会主义发展阶段的思想**
   - 毛泽东提出：社会主义可分为两个阶段——**不发达的社会主义**和**比较发达的社会主义**
   - 后一阶段可能比前一阶段需要更长的时间
9. **经济体制改革的初步探索**
   - 毛泽东：发展商品生产、利用价值规律的思想
   - 刘少奇：使社会主义经济既有计划性又有多样性和灵活性
   - 陈云：建立"适合于我国情况和人民需要的社会主义的市场"
   - 毛泽东还提出"**两参一改三结合**"（干部参加劳动，工人参加管理，改革不合理的规章制度，工人群众、领导干部和技术人员三结合）
   - 邓小平：整顿工业企业，改善和加强企业管理，实行职工代表大会制
10. **其他理论成果**
    - "四个现代化"战略目标
    - "一纲四目"（周恩来归纳的和平解决台湾问题的方针）
    - "三个世界划分"战略思想（苏美两个超级大国属于第一世界）
    - 两个"中间地带"战略思想
11. **初步探索的意义**
    - 巩固和发展了我国的社会主义制度
    - 为开创中国特色社会主义提供了宝贵经验、理论准备、物质基础
    - 丰富了科学社会主义的理论和实践
12. **初步探索的经验教训**
    - 必须把马克思主义与中国实际相结合
    - 必须正确认识社会主义社会的主要矛盾和根本任务，集中力量发展生产力
    - 必须从实际出发，建设规模和速度要和国力相适应
    - 必须发展社会主义民主，健全社会主义法制
    - 必须坚持党的民主集中制和集体领导制度
    - 必须坚持对外开放，不能关起门来搞建设

#### 必背简答题

- **党在社会主义建设道路的初步探索中取得了哪些重要的理论成果？**
- **如何认识党对社会主义建设道路初步探索的意义？**
- **党对社会主义建设道路的初步探索有哪些经验教训？**

### 【第六章】邓小平理论

#### 核心知识点

1. **邓小平理论首要的基本的理论问题**
   - **什么是社会主义，怎样建设社会主义**
2. **邓小平理论的精髓**
   - **解放思想、实事求是**
3. **社会主义本质**（1992年南方谈话）
   - **解放生产力，发展生产力，消灭剥削，消除两极分化，最终达到共同富裕**
   - "解放生产力，发展生产力"——从生产力角度
   - "消灭剥削，消除两极分化"——从生产关系角度
   - "最终达到共同富裕"——从发展目标角度
4. **社会主义初级阶段理论**
   - 党的十三大系统阐述
   - 两层含义：
     - 第一，我国社会已经是社会主义社会，必须坚持而不能离开社会主义
     - 第二，我国的社会主义社会还处在初级阶段，必须正视而不能超越
   - 是建设中国特色社会主义的**总依据**
5. **党在社会主义初级阶段的基本路线**
   - 党的十三大提出
   - **领导和团结全国各族人民，以经济建设为中心，坚持四项基本原则，坚持改革开放，自力更生，艰苦创业，为把我国建设成为富强、民主、文明的社会主义现代化国家而奋斗**
   - "一个中心、两个基本点"：以经济建设为中心，坚持四项基本原则，坚持改革开放
   - 是党和国家的**生命线**，人民的**幸福线**
6. **社会主义的根本任务**
   - **发展生产力**（注意：不是解放生产力）
   - "发展才是硬道理"
   - **科学技术是第一生产力**
7. **"三步走"发展战略**（党的十三大确定）
   - 第一步：1981-1990年，解决温饱问题
   - 第二步：1991年到20世纪末，达到小康水平
   - 第三步：到21世纪中叶，人均国民生产总值达到中等发达国家水平，基本实现现代化
8. **改革开放理论**
   - 改革是社会主义社会发展的**直接动力**
   - **"三个有利于"标准**：是否有利于发展社会主义社会的生产力，是否有利于增强社会主义国家的综合国力，是否有利于提高人民的生活水平
   - 对外开放是建设中国特色社会主义的一项基本国策
9. **社会主义市场经济理论**
   - 计划经济和市场经济**不是划分社会制度的标志**
   - 计划和市场都是**经济手段**
   - 市场经济作为资源配置方式本身**不具有制度属性**
   - 党的十四大确定建立社会主义市场经济体制的改革目标
10. **"两手抓，两手都要硬"**
    - 一手抓物质文明，一手抓精神文明
    - 一手抓建设，一手抓法制
    - 一手抓改革开放，一手抓惩治腐败
11. **"一国两制"**
    - 核心：**坚持一个中国**
    - 首先运用于解决香港、澳门问题
    - 成功解决了香港问题和澳门问题（**不包括台湾问题**，台湾问题尚未解决）
12. **时代主题**
    - **和平与发展**是当今世界两大带有全球性的战略问题
    - 并不意味着这两个问题已经解决
13. **邓小平理论的历史地位**
    - 马克思列宁主义、毛泽东思想的继承和发展
    - 中国特色社会主义理论体系的**开篇之作**
    - 改革开放和社会主义现代化建设的科学指南
14. **关键会议与文献**
    - 十一届三中全会：伟大转折，改革开放
    - 1978年《解放思想，实事求是，团结一致向前看》：开辟新时期新道路的宣言书
    - 党的十二大：邓小平正式提出"建设有中国特色的社会主义"
    - 党的十三大：系统阐述社会主义初级阶段理论，提出基本路线，"三步走"战略
    - 南方谈话（1992年）：邓小平理论走向成熟的标志
    - 党的十四大：确立社会主义市场经济体制目标
    - 党的十五大：把邓小平理论确立为党的指导思想

## 三、老师课堂追问重点（高频考点）

以下为录音中老师反复提问、强调必须掌握的内容：

1. **马克思主义中国化什么时候提出？** → 1938年10月，六届六中全会，《论新阶段》
2. **马克思主义中国化经历了几次飞跃？** → 三次（毛泽东思想是第一次；中国特色社会主义理论体系是新的飞跃；习近平新时代中国特色社会主义思想是新的飞跃）
3. **毛泽东思想活的灵魂是什么？** → 实事求是、群众路线、独立自主
4. **三大法宝是什么？各有什么内涵？之间什么关系？**
5. **毛主席最大的贡献是什么？** → 找到了一条道路：**农村包围城市、武装夺取政权**
6. **新民主主义革命"新"在哪？** → 领导阶级（无产阶级）、指导思想（马克思主义）、属于世界无产阶级革命一部分
7. **新旧民主主义革命的共同特征？** → 都是**资产阶级民主主义革命**
8. **新民主主义革命的敌人是谁？** → 三座大山（帝国主义、封建主义、官僚资本主义），最凶狠的是帝国主义
9. **革命的朋友/动力有哪些？** → 四个阶级：工人阶级、农民阶级、城市小资产阶级、民族资产阶级；最主要依靠工农联盟
10. **城市小资产阶级包括哪些人？** → 知识分子、小商人、手工业者、自由职业者
11. **民族资产阶级有什么特征？** → **两面性**：革命性 + 妥协性（墙头草）
12. **1949年10月1日成立的是什么国家？** → 新民主主义共和国（存在到1956年底）
13. **过渡时期总路线是什么？** → **一化三改**（工业化 + 对农业、手工业、资本主义工商业的改造）
14. **如何正确评价毛泽东及其毛泽东思想？**
15. **毛泽东思想形成各时期的代表著作？**
16. **长征找到了一条什么道路？** → 今年是长征90周年，相关知识点要重点复习

## 四、复习建议

1. **客观题（50分）**：全面复习每章每节的单选题、多选题、判断题，利用题库反复练习

6. **课后思考题**：对照课后思考题逐一弄明白
